What is HTML?

It's a language that defines the structure of content on a web page. The syntax is based on SGML (Standard Generalized Markup Language) which used opening and closing tags to provide meaning to unorganized text. It can also handle video, images, audio, and canvases for graphics. Inside the tags is the content. It can also nest more tags inside tags (children).
